# KintsugiOps AI 3-Minute Demo Script

## 1. Problem

Software teams waste money, energy, and engineering time through over-provisioned cloud resources, bloated containers, redundant CI pipelines, unused dependencies, idle workloads, and repeated AI API calls. Most tools report the waste, but they do not behave like a repair team.

## 2. Cultural Inspiration

Kintsugi is the Japanese art of repairing broken pottery with gold. The repair is visible, intentional, and valuable. KintsugiOps AI applies that idea to software: the cracks are waste findings, the gold repairs are optimizations, and the restored vessel is a more efficient digital system.

## 3. Product Demo

I start by selecting a sample audit package called Bloated SaaS backend. It includes package metadata, a Dockerfile, CI workflow, billing CSV, AI usage log, architecture notes, and a repository tree. The app works without API keys because the demo has deterministic mock agent outputs.

## 4. Agent Workflow

Now the autonomous agents run. The Crack Finder Agent detects software waste signals. The Cloud Waste Agent finds right-sizing and idle compute opportunities. The Carbon Accountant Agent estimates kgCO2e. The Repair Planner Agent ranks fixes using impact times confidence divided by effort and risk. The Risk Verifier Agent adds approval gates. Finally, the Impact Report Agent prepares a stakeholder-ready repair ledger.

## 5. X402 Payment Moment

The X402 Payment Agent tries to call a paid Carbon Intensity API. The API returns HTTP 402 Payment Required. The agent reads the price, checks its budget, approves the payment, sends a mock X-PAYMENT header, receives the premium audit result, and logs a receipt. This is simulated for hackathon reliability, but the code is structured so a real X402 integration can replace it.

## 6. Business and Sustainability Impact

The dashboard estimates monthly cost waste, annual savings, carbon reduction, engineering hours saved, and a Kintsugi Score improvement. For the main sample, the app shows a Docker image reducible from 1.8GB to 650MB, CI runtime reducible from 24 minutes to 11 minutes, 38% avoidable AI API calls, 17 idle compute hours per day, and more than $18,000 in annual savings.

## 7. Closing Line

KintsugiOps AI turns broken, wasteful software systems into repaired, measurable, and more sustainable digital infrastructure.
